# 🏝️ Chapter 1 - Scene 3: 화물칸 (위험물 적재소)

> "코를 찌르는 매캐한 화학 약품 냄새가 진동합니다... 치명적인 산성 웅덩이가 길을 막고 있습니다."

위험한 화학 물질과 전력 퍼즐이 있는 곳입니다. 안전 장비를 만들고 도구를 확보해야 합니다.

---

## 👁️ 시야 내 상호작용 요소 (전체 목록)

이 씬에서 입력하여 조사할 수 있는 **모든 키워드**입니다.

* **주요 설비**: `작업대`, `전자 금고`, `소방 도끼` (벽에 고정됨), `녹슨 클램프`
* **화학 물질**: `산성 웅덩이`, `하얀 가루`, `흩어진 보급품`
* **단서**: `정비 메모`, `주기율표`, `수수께끼 쪽지`, `멀티미터`

---

## 💡 단계별 진행 가이드 (상황별 힌트)

### 1단계: 산성 웅덩이 건너기
강산성 웅덩이가 길을 막고 있습니다. 중화시켜야 하지만, 그전에 나중을 위해 샘플을 챙겨야 합니다.
1.  **`흩어진 보급품`**을 뒤져 **`전분 가루`**, **`건전지`** 등을 확보하세요.
2.  **`녹슨 양동이`**로 **`하얀 가루`**(가성소다)를 퍼담으세요. (입구가 넓은 용기 필요)
    * 입력 예시: `녹슨 양동이 + 하얀 가루`
3.  **`빈 페트병`**으로 **`산성 웅덩이`**의 샘플(**`산성 용액 병`**)을 먼저 채취하세요.
    * 입력 예시: `빈 페트병 + 산성 웅덩이`
4.  이제 **`가성소다 양동이`**를 **`산성 웅덩이`**에 부어 길을 여세요.
    * 입력 예시: `가성소다 양동이 + 산성 웅덩이`

### 2단계: 전자 금고 열기 (배터리 퍼즐)
금고의 전원을 켜야 합니다. **`정비 메모`**에 적힌 **19V**를 맞춰야 합니다.
1.  **`멀티미터`**로 각 **`건전지`**의 전압을 측정하세요. (9V, 6V, 5V, 4V, 2V 건전지가 섞여 있습니다.)
2.  합이 **19V**가 되도록 세 개를 골라 **`배터리 케이스`**에 넣으세요. (메모 힌트: 높은 전압부터 순서대로)
    * 정답 조합: 9V + 6V + 4V = 19V
    * 입력 예시: `배터리 케이스 : 124` (건전지 번호 순서)
3.  완성된 **`배터리 팩`**을 **`전자 금고`**에 연결하세요.
    * 입력 예시: `배터리 팩 + 전자 금고`
4.  금고가 켜지면 **`수수께끼 쪽지`**와 **`주기율표`**를 참고해 비밀번호를 입력하세요. (금, 은, 구리의 원자 번호)
    * 힌트: 금(Au)=79, 은(Ag)=47, 구리(Cu)=29
    * 입력 예시: `전자 금고 : 794729`
5.  금고에서 **`산업용 배터리`**를 획득합니다.

### 3단계: 소방 도끼 획득 (화학 젤)
벽에 있는 **`소방 도끼`**는 녹슨 클램프에 물려 있습니다. 액체 산성 용액을 그냥 부으면 흘러내리므로, 끈적하게 만들어야 합니다.
1.  **`산성 용액 병`**과 **`전분 가루`**를 섞어 **`산성 젤`**을 만드세요.
    * 입력 예시: `산성 용액 병 + 전분 가루`
2.  **`산성 젤`**을 **`녹슨 클램프`**에 발라 녹을 제거하세요.
    * 입력 예시: `산성 젤 + 녹슨 클램프`
3.  **`소방 도끼`**를 획득하고 밖으로 나가세요.
