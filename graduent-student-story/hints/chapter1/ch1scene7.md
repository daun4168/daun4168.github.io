# 🏝️ Chapter 1 - Scene 7: 지하 호수 (발전기 가동)

> "머리 위 동굴 천장에서 흰 물줄기가 폭포처럼 쏟아져 내려... 여기서 물을 전기로 바꿔 먹을 수만 있다면..."

직접 수력 발전기를 만들어 배터리를 충전해야 하는 하이라이트 구간입니다. 지금까지 모은 재료들을 조합하세요.

---

## 👁️ 시야 내 상호작용 요소 (전체 목록)

이 씬에서 입력하여 조사할 수 있는 **모든 키워드**입니다.

* **환경**: `폭포`, `지하 호수`, `호숫가`, `검은 바위` (자철석)
* **장치**: `수력 발전기` (설치 후)
* **이동**: `경사로` (석회 동굴로 복귀)

---

## 💡 단계별 진행 가이드 (상황별 힌트)

### 1단계: 발전기 부품 준비
먼저 필요한 재료를 모두 모으고 가공해야 합니다.
1.  **`호숫가`**를 조사해 **`바위 고리`**와 **`나무 축`**을 얻으세요.
2.  **`검은 바위`**를 **`소방 도끼`**로 내려쳐 **`자철석 조각`**을 얻으세요.
    * 입력 예시: `소방 도끼 + 검은 바위`

### 2단계: 발전기 모듈 조립
인벤토리 안에서 부품들을 단계적으로 조합하세요.
1.  **수차 로터**: `코팅된 코코넛 껍질` + `덩굴 줄기` -> **`임시 수차 로터`**.
2.  **축 연결**: `임시 수차 로터` + `나무 축` -> **`축 달린 수차`**.
3.  **베어링 결합**: `축 달린 수차` + `바위 고리` -> **`고리에 끼운 수차`**.
4.  **코어 제작**: `자철석 조각` + `절연 구리선` -> **`발전 코어`**.
5.  **모듈 완성**: `고리에 끼운 수차` + `발전 코어` -> **`수력 발전 모듈`**.

### 3단계: 설치 및 가동
1.  완성된 **`수력 발전 모듈`**을 **`폭포`**에 설치하세요.
    * 입력 예시: `수력 발전 모듈 + 폭포`
2.  설치된 **`수력 발전기`**의 잠금을 해제해야 합니다. 검은 바위 묘사(노랑-4, 보라-7, 빨강-2)가 힌트입니다.
    * 입력 예시: `수력 발전기 : 472`
3.  **`산업용 배터리`**를 연결해 충전하세요.
    * 입력 예시: `산업용 배터리 + 수력 발전기`
4.  **`충전된 산업용 배터리`**를 획득했다면 다시 위로 올라가 **`절벽 길`**로 향하세요.
